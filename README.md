# Salotti Request

Este projeto tem como objetivo praticar HTML, CSS e JavaScript,
além de trabalhar com versionamento em repositório compartilhado.

## Tarefas

Os alunos deverão corrigir e implementar melhorias no projeto.

### 1 - Estilos do botão success
Alterar o estilo da classe "success":
- Cor de fundo verde
- Cor do texto branca ou preta (o que ficar melhor visualmente)

### 2 - Corrigir o querySelector
Existe um erro no arquivo script.js.
O botão não está sendo encontrado corretamente.
Corrija este bug.

### 3 - Variáveis CSS
Criar variáveis CSS para cores no :root e substituir todas as cores
escritas manualmente pelas variáveis.

Exemplo:
:root {
    --cor-verde: #28a745;
}
